# Assets Directory

## Logo

Place your logo file here:
- **File**: `logo.pdf` (or convert to `logo.png`/`logo.jpg`)
- **Usage**: The logo is referenced in:
  - `components/Navigation.tsx`
  - `components/Hero.tsx`
  - `components/Footer.tsx`

## Instructions

1. Add your logo file to this directory
2. Update the logo references in the components mentioned above
3. If using a PDF, consider converting to PNG/JPG for web use:
   - Use an online converter or design tool
   - Recommended size: 200x200px minimum for high-quality display
   - Transparent background preferred

## Color Extraction

To extract colors from your logo for the Tailwind config:
1. Open your logo in a design tool (Figma, Photoshop, etc.)
2. Identify primary and accent colors
3. Update `tailwind.config.js` with your brand colors
